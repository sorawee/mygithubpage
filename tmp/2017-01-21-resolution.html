Contradiction

In logic, contradiction is denoted by a symbol $\perp$this symbol is called falsum. We can view $\perp$ as the same as $p \and \neg p$ because the formula is a contradiction: it evaluates to false in all truth assignments.

Here are some facts that are useful:

  1. An empty clause (disjunction of empty set) is $\perp$. We have seen this already in the SAT lab.
  2. $\neg p$ which states “it is not the case that $p$,” is logically equivalent to $p \to \perp$. The latter basically states “if you have $p$, then you have a contradiction.” This perfectly captures the meaning of “it is not the case that $p$" because if it is indeed that “it is not the case that $p$,” we should not be able to have $p$, so we will not have a contradiction.

Resolution Rule

Given a CNF formula $\phi$, we would like to know if $\phi$ is satisfiable or not. Asking if $\phi$ is satisfiable is the same as asking if $\phi$ is unsatisfiable and then negate the answer. If $\phi$ is unsatisfiable, then $\neg \phi$, or equivalently $\phi \to \perp$, is a tautology. Therefore, with an expressive deduction system, $\phi \to \perp$ should be provable. Equivalently, given $\phi$, we should be able to derive or prove $\perp$.

The deduction system that we are going to use has one inference rule called resolution rule. It states that, if we have:

  • $C \or X$ and
  • $\neg C \or Y$

Then we can derive $X \or Y$, where $C$ is a variable and $X$ and $Y$ are any formulas. To see why this is soundsoundness means, things that we get from derivation is correct, notice that $C \or \neg C$ is a tautology which states: “it is the case that $C$, or it is the case that $\neg C$.”

  • If it is the case that $C$: we have already known that it is the case that $\neg C \or Y$, but it is not the case that $\neg C$, so it is the case that $Y$
  • If it is the case that $\neg C$: we have known that it is the case that $C \or X$, but it is not the case that $C$, so it is the case that $X$

So we are able to say that it is the case that $X \or Y$.

By the completness of propositional logic, this is also the same as asking if it is possible to derive a contradition using deduction rules.

Because $\phi$ is logically equivalent to $\varphi$, to ask if $\phi$ is satisfiable is to if $\varphi$ is satisfiable. Another closely related notion of satisfiability is logical consistency: a set is logically consistent if there is at least one truth assignment to make all formula in the set to evaluate to $T$. The set is logically inconsistent otherwise. Notice that our original goal is the same as asking if the set $S$ below is logically consistent or not:

$$S = \set{(A_{1,1} \or A_{1,2} \or ...), (A_{2,1} \or A_{2,2} \or ...), ..., (A_{m,1} \or A_{m,2} \or ...)}$$

Because $S$ is either logically consistent or logically inconsistent, what we will do is trying to deduce a contradiction from $S$ which is a proof that $S$ is logically inconsistent. If we could do so, then $S$ is not logically consistent, so $\phi$ is not satisfiable. On the other hand, if it is impossible to deduce a contradiction from $S$, then $S$ is consistent, so $\phi$ is satisfiable.
