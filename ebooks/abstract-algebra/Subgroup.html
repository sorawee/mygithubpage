3 Subgroup

3.1 Definition of Subgroup

Like sets, which have subsets, groups have substructures too.

Definition : \(H\) is a subgroup of \(G\) (denoted by \(H \le G\)) iff

  • \(H \subseteq G\)

  • \(H\) is a group with the operation defined in \(G\)

3.2 The Subgroup Criterion

To prove that \(H\) is a subgroup of \(G\), we need to check that \(H\) is a subset of \(G\), and it is a group (satisfying G0, G1, G2, and G3). This is a lot of work. It would be nice if we don’t have to check all properties. In particular, we can use the fact that \(H\) needs to be a subset of \(G\) and will have the same operation as \(G\) to reduce our check. For example, there is no need to check for G3 at all because if \(G\) is associative, then the operation on elements in \(H\) which are a part of \(G\) will be associative as well. The following is a reduced criterion that we need to check to be sure that \(H \le G\).

Proposition (The Subgroup Criterion): \(H\) is a subgroup of \(G\) iff

  • \(H \subseteq G\)

  • \(H \ne \emptyset\)

  • \(\forall a, b \in H. a * b^{-1} \in H\)

Proof : As usual, to prove "if and only if", we need to prove for both directions:

(\(\rightarrow\)) First, we need to prove that if \(H \le G\), then

  • \(H \subseteq G\)

  • \(H \ne \emptyset\)

  • \(\forall a, b \in H. a * b^{-1} \in H\)

Because \(H \le G\), by definition of subgroup, \(H \subseteq G\)

Because \(H\) is a subgroup, \(H\) is also a group. Thus, \(e \in H\), showing that \(H \ne \emptyset\).

Because \(H\) is a group, \(\forall c \in H. c^{-1} \in H\) and \(\forall a, b \in H. a*b \in H\). Applying \(\forall a, b \in H. a*b \in H\) by taking \(b\) to be \(c^{-1}\), we obtain \(\forall a, c \in H. a * c^{-1} \in H\). That is, \(\forall a, b \in H. a * b^{-1} \in H\).

These conclude (\(\rightarrow\)).

(\(\leftarrow\)) Second, we need to prove that if

  • \(H \subseteq G\)

  • \(H \ne \emptyset\)

  • \(\forall a, b \in H. a * b^{-1} \in H\)

then \(H \le G\)

We are more interested in proving this direction, because it lets us conclude that \(H \le G\).

To show that \(H \le G\), we need to show that \(H \subseteq G\) and \(H\) is a group. The first criterion directly corresponds to the first condition.

Now, we need to prove that \(H\) is a group by showing that \(H\) satisfies G0, G1, G2, and G3.

To show that \(e \in H\), we notice that \(H \ne \emptyset\), so there is an element \(z \in H\). Applying \(\forall a, b \in H. a * b^{-1} \in H\) by taking both \(a\) and \(b\) to be \(z\), we obtain \(z * z^{-1} = e \in H\).

To show that \(\forall a \in H. a^{-1} \in H\), we notice that \(e \in H\) (and we just prove this in the previous paragraph). Applying \(\forall a, b \in H. a * b^{-1} \in H\) by taking \(a\) to be \(e\), we obtain \(\forall b \in H. e * b^{-1} = b^{-1} \in H\). That is, \(\forall a \in H. a^{-1} \in H\).

To show that \(\forall a, b \in H. a * b \in H\), we notice that \(\forall c \in H. c^{-1} \in H\) (and we just prove this in the previous paragraph). Applying \(\forall a, b \in H. a * b^{-1} \in H\) by taking \(b\) to be \(c^{-1}\), we obtain \(\forall a, c \in H. a * (c^{-1})^{-1} = a * c \in H\). That is, \(\forall a, b \in H. a * b \in H\).

To show that \(\forall a, b, c \in H. (a * b) * c = a * (b * c)\), we notice that because \(G\) is a group, \(\forall a, b, c \in G. (a * b) * c = a * (b * c)\). Since \(H \subseteq G\), it follows that \(\forall a, b, c \in H. (a * b) * c = a * (b * c)\) as well.

These conclude (\(\leftarrow\)) and thus the proposition.

3.3 Set Operations on Subgroups

Because groups are sets, they can interact with each other using set operations like intersection or union. However, it does not make much sense to perform operations on groups with different operations. Thus, we are more interested to perform set operations on subgroups since they all have the same operation and the operation is defined for all pairs of elements of subgroups.

It turns out that a union of two subgroups is not interesting much: it does not have a special property – in particular, it is not always a group. An example is, let \(A = \{0, 2, 4\}, B = \{0, 3\}, G = \mathbb{Z}_6\) be groups with operation \(+_6\). Then, \(A \le G\) and \(B \le G\), but the operation does not close over \(A \cup B = \{0, 2, 3, 4\}\) because \(2 +_6 3 = 5 \not\in A \cup B\). Thus, \(A \cup B\) is not a group.

However, surprisingly, an intersection of two subgroups turns out to be a group. Moreover, it is also a subgroup too!

Theorem : If \(H_1\) and \(H_2\) are subgroups of \(G\), then \(H_1 \cap H_2\) is also a subgroup of \(G\).

Proof : To verify that \(H_1 \cap H_2 \le G\), we need to show that it satisfies the subgroup criterion.

Proposition (The Subgroup Criterion): \(H\) is a subgroup of \(G\) iff

  • \(H \subseteq G\)

  • \(H \ne \emptyset\)

  • \(\forall a, b \in H. a * b^{-1} \in H\)

For the first one, because \(H_1\) is a subgroup of \(G\), by definition, \(H_1 \subseteq G\). Combining this with the fact that \(H_1 \cap H_2 \subseteq H_1\), we obtain \(H_1 \cap H_2 \subseteq H_1 \subseteq G\). By transitivity of subset, we conclude that \(H_1 \cap H_2 \subseteq G\).

For the second one, because \(H_1\) is a group, by its definition, \(e \in H_1\). Similarly, \(e \in H_2\). Thus, \(e \in H_1 \cap H_2\) and \(H_1 \cap H_2 \ne \emptyset\)

For the third one, because \(H_1 \cap H_2 \subseteq H_1\), by the definition of subset \(\forall a, b \in H_1 \cap H_2. a, b \in H_1\). Since \(H_1\) is a group, \(a * b^{-1} \in H_1\). Using the same argument with \(H_2\), it follows that \(a * b^{-1} \in H_2\) as well. Thus, \(a * b^{-1} \in H_1 \cap H_2\).

With these, we can conclude that \(H_1 \cap H_2 \le G\).
