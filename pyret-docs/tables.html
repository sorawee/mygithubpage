3.5 Tables

Table

The type of a table.

A table can be constructed by the table syntax.

Examples:
my-table = table: col1 :: Number, col2 :: String
    row: 1, "foo"
    row: -123, "bar"
  end

The above example will construct a table with two columns: col1 :: Number and col2 :: String. Please note that the order of columns matters. That is, 
table: col1 :: Number, col2 :: String
is not the same as 
table: col2 :: String, col1 :: Number
.
