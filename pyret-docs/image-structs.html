4.8 image-structs

Usage:
import image-structs as ...

4.8.1 Data types

data Color:
| color(red :: Number, green :: Number, blue :: Number, alpha :: Number)
end

color :: (
red :: Number,
green :: Number,
blue :: Number,
alpha :: Number
)
-> Color

Valid arguments are in the range 0–255, inclusive.
